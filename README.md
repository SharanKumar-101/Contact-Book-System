# Contact Book System

A simple, text-based contact management application written in Python. This project was built to manage contact details efficiently.

## Features
* **Add Contacts:** Save names, phone numbers, and emails.
* **Search:** Quickly look up contact information by name.
* **Display All:** View all saved contacts in a clean list layout.
* **Update:** Easily modify a contact's phone number.
* **Delete:** Remove unwanted contacts instantly.

## How to Run
1. Make sure you have Python installed on your computer.
2. Download the `Contact.py` file.
3. Open your terminal or command prompt, navigate to the folder, and run:
```bash
   python Contact.py
