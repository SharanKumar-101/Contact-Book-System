# Contact Book System

A simple, text-based contact management application written in Python[cite: 4]. This project was built as an Activity-Based Assessment to manage contact details efficiently[cite: 4, 6].

## Features
* **Add Contacts:** Save names, phone numbers, and emails[cite: 4].
* **Search:** Quickly look up contact information by name[cite: 4].
* **Display All:** View all saved contacts in a clean list layout[cite: 4].
* **Update:** Easily modify a contact's phone number[cite: 4].
* **Delete:** Remove unwanted contacts instantly[cite: 4].

## How to Run
1. Make sure you have Python installed on your computer.
2. Download the `PythonABA.py` file[cite: 2].
3. Open your terminal or command prompt, navigate to the folder, and run:
```bash
   python PythonABA.py
